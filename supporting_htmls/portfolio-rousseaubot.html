RousseauBot on Twitter
Curious about how Twitter bots worked, I set out to create one myself - I use to see them running amuck tweeting to famous people like Elon Musk. After some research, I found that the first step is to set up an account for your bot and then apply for Twitter Developers access to that account's back-end. This process is fairly straightforward and quick within the Twitter Developers portal - just remember to note down all your API/client tokens and keys!

Of course, I didn't want to create anything as aggressive as those bots that harass everyone's tweet feeds; I aimed to make a bot paying hommage to one of my favourite authors. I first needed to grab some Rousseau source material. I found a page on the Online Library of Liberty website containing a piece by Rousseau that I liked, On the Social Contract. Using BeautifulSoup and the requests Python libraries, I scraped the webpage for text (using regex to split paragraphs into sentences and to filter for "good" sentences to use e.g. ignore "sentences" that are really just chapter titles). This is all then stored in a txt file for a separate script to read.

I wrote another program that read a randomly-selected quote from the txt file above and posted it to RousseauBot's account on Twitter (as shown on the right - go follow!), keeping track of the quotes tweeted over the past 10 days to avoid short-term repetition in a deterministic manner.
